# vision@home

ROS + Tensorflow Object Detection API

## Installation

#### Install OpenCV

[OpenCV Tutorial](http://docs.opencv.org/2.4/doc/tutorials/introduction/linux_install/linux_install.html)

#### Run tf object detection script

```shell
chmod +x install-tf-api.sh
./install-tf-api.sh
```
